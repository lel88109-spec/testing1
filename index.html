About Us

About Cloud Classifier

Cloud Classifier is an AI-powered web application that helps users identify and understand different types of clouds using machine learning technology. Built with TensorFlow and Teachable Machine, our tool makes cloud classification accessible to everyone.

Whether you're a student, weather enthusiast, or researcher, our intuitive interface provides real-time cloud identification and weather insights.
